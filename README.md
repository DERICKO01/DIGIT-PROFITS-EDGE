## Welcome to DIGIT PROFIT EDGE

**DIGIT PROFIT EDGE** is a professional financial markets education and trading platform designed to help traders develop the knowledge, skills, and discipline required to succeed in today's financial markets.

Whether you're a complete beginner or an experienced trader, DIGIT PROFIT EDGE provides structured learning, market analysis, mentorship, and trading tools to support your trading journey.

# Installation

1. Clone the repository.

```bash
git clone https://github.com/yourusername/digit-profit-edge.git
```

2. Open the project folder.

```bash
cd digit-profit-edge
```

3. Open **index.html** in your preferred browser.

Or run using VS Code Live Server.
